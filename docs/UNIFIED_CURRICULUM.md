# 统一课程总纲：Learning Python with AI IDE（含实践案例复现）

本课程只有一条主线：学习 Python。Stanford CS146S 作为实践案例，用来复现实战代码与理念，帮助学员把基础能力落到可运行的项目上。

## 1. 学习理念
- **掌控而不是替代**：AI 可以执行，但你要能判断它做了什么、做没做对。
- **可复现优先**：每一步都能复现，才算真正学会。
- **先理解再加速**：先掌握基础，再让 AI 放大效率。

## 2. 主线叙事
- **工具链入门**：终端、环境、IDE、脚本运行（Week1 基础）。
- **提示词与可控输出**：通过实践案例完成 Week1 prompting 复现。
- **工程化与项目化**：以 Stanford Week2-Week8 案例为骨架，主线补齐基础（待补齐）。

## 3. 课程组织方式
- 每周包含 3 个层次：
  - **主线 Lesson**：零基础的操作与解释。
  - **实践案例复现**：对照官方代码完成提示词/脚本任务（案例目录：`curriculum/Case Studies/Stanford CS146S/`）。
  - **产出与验收**：有结果、可复盘。
- 官方代码只引用 `references/office-code/`，不在课程中直接修改。
