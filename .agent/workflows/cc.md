---
description: Run a continuity check on a lesson file to ensure titles and flow are consistent.
---

Usage: `/check-continuity <lesson-file-path>`

Steps:
1. Verify the file header matches the filename.
2. Check the previous and next lesson files to ensure titles and transitions are consistent.
3. Verify that all terminology follows the regional standards (SC/TC).
4. Update any mismatched references.
