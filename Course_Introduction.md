## 🇨🇳 简体中文版：回归本质的工程进化之路

欢迎加入《Learn Python with AI IDE》。这不仅仅是一门 Python 编程课，更是一套响应 AI 范式革命的工程训练体系。我们拒绝低幼化的教育模式和虚浮的文学隐喻，坚持以**严肃、硬核、专业**的工程师态度，带你构建跨越多个维度的核心竞争力。

### 核心体系：双主轨架构
本课程体系采用“基础构建”与“实战演练”深层嵌套的双主轨设计：

1. **基础主轨 (Module-based)**：
   - **Module 1 (The AI-Ready Engineer)**：从系统终端与隔离环境切入，建立工程底座。你将学会掌控终端、理解解释器逻辑，并在 Antigravity IDE 中建立基于“授权”的自动化协作协议。
   - **Module 2 (Python Essentials)**：在 AI 辅助下全量对齐 Andrew Ng 课程的核心知识点。从变量、数据结构到文件操作，重点不再是记忆语法，而是理解逻辑链条与调试能力。
   - **未来进阶**：涵盖 API 集成、自动化脚本、数据流处理等现代开发核心。

2. **实战主轨 (CS146S Stanford Series)**：
   遵循斯坦福 CS146S 项目制逻辑，将基础 Module 学到的技能即时应用于 Agent 构建、RAG 检索增强、向量数据库等前沿 AI 应用实战中。

### 我们的教学理念
- **工具主权 (Tool Sovereignty)**：在 Antigravity 中，你和 AI 的关系是“授权者与执行者”。AI 可以自动化创建文件、运行命令，但每一次关键操作都必须经过你的 **Approve**（授权）。主权不在于亲手敲代码，而在于掌握“允许它发生”的权力。
- **工程成熟度 (Engineering Maturity)**：比代码更重要的是工程直觉。我们强调环境隔离（虚拟环境）、路径感知（绝对/相对路径）与版本控制，确保你的作品是经得起推敲的工业级系统。
- **底层透视 (Raw Logic)**：我们不掩盖复杂性。即使 AI 在从事实时自动化，我们也会引导你观察“影子终端”中的真实命令。只有理解底层，才能驾驭顶层。

在这里，你不仅是学习 Python，更是在重塑自己在 AI 时代的职业身份——成为那名能够驾驭智能工具、解决复杂难题的 **AI-Ready Engineer**。

## 🇹🇼 繁體中文版：回歸本質的工程進化之路

歡迎加入《Learn Python with AI IDE》。這不僅僅是一門 Python 程式設計課，更是一套回應 AI 範式革命的工程訓練體系。我們拒絕低幼化的教育模式和虛浮的文學隱喻，堅持以**嚴肅、硬核、專業**的工程師態度，帶你構建跨越多個核心維度的競爭力。

### 核心體系：雙主軌架構
本課程體系採用「基礎構建」與「實習演練」深層嵌套的雙主軌設計：

1. **基礎主軌 (Module-based)**：
   - **Module 1 (The AI-Ready Engineer)**：從系統終端機與隔離環境切入，建立工程底座。你將學會掌控終端機、理解直譯器邏輯，並在 Antigravity IDE 中建立基於「授權」的自動化協作協議。
   - **Module 2 (Python Essentials)**：在 AI 輔助下全量對應 Andrew Ng 課程的核心知識點。從變數、資料結構到檔案操作，重點不再是記憶語法，而是理解邏輯鏈條與除錯能力。
   - **未來進階**：涵蓋 API 整合、自動化腳本、資料串流處理等現代開發核心。

2. **實戰主軌 (CS146S Stanford Series)**：
   遵循史丹佛 CS146S 專案制邏輯，將基礎 Module 學到的技術即時應用於 Agent 構建、RAG 檢索增強、向量資料庫等前沿 AI 應用實務中。

### 我們的教學理念
- **工具主權 (Tool Sovereignty)**：在 Antigravity 中，你和 AI 的關係是「授權者與執行者」。AI 可以自動化創建檔案、執行指令，但每一次關鍵操作都必須經過你的 **Approve**（授權）。主權不在於親手敲代碼，而在於掌握「允許它發生」的權力。
- **工程成熟度 (Engineering Maturity)**：比程式碼更重要的是工程直覺。我們強調環境隔離（虛擬環境）、路徑感知（絕對/相對路徑）與版本控制，確保你的作品是經得起推敲的工業級系統。
- **底層透視 (Raw Logic)**：我們不掩蓋複雜性。即使 AI 在從事實時自動化，我們也會引導你觀察「影子終端機」中的真實指令。只有理解底層，才能駕馭頂層。

在這裡，你不僅是學習 Python，更是在重塑自己在 AI 時代的職業身份 —— 成為那名能夠駕馭智慧工具、解決複雜難題的 **AI-Ready Engineer**。
