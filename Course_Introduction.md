> **From "Black Box User" to "Versatile AI Engineer": Master the Synergy of Code and AI**

## 🇨🇳 简体中文版：双轨并行的工程进化之路

欢迎加入《Learn Python with AI IDE》。这不仅仅是一门 Python 编程课，更是一套响应 AI 范式革命的工程训练体系。我们拒绝低幼化的教育模式，坚持以**严肃、硬核、专业**的工程师态度，带你构建跨越多个维度的核心竞争力。

### 核心体系：双主轨架构
本课程体系采用“基础构建”与“实战演练”深层嵌套的双主轨设计：

1. **基础主轨 (Module-based)**：
   - **Module 1 (The AI-Ready Engineer)**：从系统终端与隔离环境切入，建立工程底座。你将学会掌控终端、理解解释器逻辑，并在 AI IDE（Antigravity）中建立专业的人机协作协议。
   - **Module 2 (Python Essentials)**：在 AI 辅助下全量对齐 Andrew Ng 课程的核心知识点。从变量、数据结构到文件操作，重点不再是记忆语法，而是理解逻辑链条与调试能力。
   - **未来进阶**：涵盖 API 集成、自动化脚本、数据流处理等现代开发核心。

2. **实战主轨 (CS146S Stanford Series)**：
   遵循斯坦福 CS146S 项目制逻辑，将基础 Module 学到的技能即时应用于 Agent 构建、RAG 检索增强、向量数据库等前沿 AI 应用实战中。

### 我们的教学理念
- **代码主权 (Code Ownership)**：AI 是你的副驾驶（Copilot），但你必须是拥有最终决定权的指挥官。看得懂、测得过、敢发布，对每一行代码负最终责任。
- **工程成熟度 (Engineering Maturity)**：比代码更重要的是工程直觉。我们强调环境隔离、版本控制与模块化思维，确保你的作品是工业级的系统，而非零散的代码碎片。
- **逻辑驱动学习**：借鉴 Stanford 严谨性与 DeepLearning.AI 的实战感，将 Prompt Engineering 与 Python 实操深度融合。

在这里，你不仅是学习 Python，更是在重塑自己在 AI 时代的职业身份——成为那名能够驾驭智能工具、解决复杂难题的 **AI-Ready Engineer**。

## 🇹🇼 繁體中文版：雙軌並行的工程進化之路

歡迎加入《Learn Python with AI IDE》。這不僅僅是一門 Python 程式設計課，更是一套回應 AI 範式革命的工程訓練體系。我們拒絕低幼化的教育模式，堅持以**嚴肅、硬核、專業**的工程師態度，帶你構建跨越多個核心維度的競爭力。

### 核心體系：雙主軌架構
本課程體系採用「基礎構建」與「實習演練」深層嵌套的雙主軌設計：

1. **基礎主軌 (Module-based)**：
   - **Module 1 (The AI-Ready Engineer)**：從系統終端機與隔離環境切入，建立工程底座。你將學會掌控終端機、理解直譯器邏輯，並在 AI IDE（Antigravity）中建立專業的人機協作協議。
   - **Module 2 (Python Essentials)**：在 AI 輔助下全量對應 Andrew Ng 課程的核心知識點。從變數、資料結構到檔案操作，重點不再是記憶語法，而是理解邏輯鏈條與除錯能力。
   - **未來進階**：涵蓋 API 整合、自動化腳本、資料串流處理等現代開發核心。

2. **實戰主軌 (CS146S Stanford Series)**：
   遵循史丹佛 CS146S 專案制邏輯，將基礎 Module 學到的技術即時應用於 Agent 構建、RAG 檢索增強、向量資料庫等前沿 AI 應用實務中。

### 我們的教學理念
- **程式碼主權 (Code Ownership)**：AI 是你的副駕駛（Copilot），但你必須是擁有最終決定權的指揮官。看得懂、測得過、敢發布，對每一行程式碼負最終責任。
- **工程成熟度 (Engineering Maturity)**：比程式碼更重要的是工程直覺。我們強調環境隔離、版本控制與模組化思維，確保你的作品是工業級的系統，而非零散的程式碼碎片。
- **邏輯驅動學習**：借鑒 Stanford 嚴謹性與 DeepLearning.AI 的實戰感，將 Prompt Engineering 與 Python 實作深度融合。

在這裡，你不僅是學習 Python，更是在重塑自己在 AI 時代的職業身份 —— 成為那名能夠駕馭智慧工具、解決複雜難題的 **AI-Ready Engineer**。
