在上一課 [M1L2] 中，我們掌握了終端機的使用。現在，我們將用終端機做一件真正像程式設計師做的事：喚醒 Python。

對於初學者來說，Python 不僅僅是一門程式語言，透過 **REPL**（Read-Eval-Print Loop），它更像是一個隨叫隨到的「超級助手」。你不需要寫長篇大論的程式碼，只需要像聊天一樣發一條指令，它就會立刻回覆你。

## 1. 第一步：環境自檢與安裝

在開始之前，我們需要確認你的電腦裡是否已經住著這隻「巨蟒」。

### 1.1 檢查是否存在
打開終端機，輸入：
```bash
# Mac 用戶：
python3 --version

# Windows 用戶：
python --version
```
*   **成功**：看到 `Python 3.x.x`。
*   **失敗**：`command not found` 或無反應。

### 1.2 安裝指南 (如果沒有)

#### 🍎 macOS 用戶
1.  **推薦方法**：訪問 [python.org](https://www.python.org/downloads/) 下載最新的 `macOS 64-bit universal2 installer`。
2.  **安裝**：雙擊 `.pkg` 檔案，一路「繼續」。
3.  **關鍵動作**：安裝完成後，**必須關閉並重啟終端機**，才能讓新安裝生效。

#### 🪟 Windows 用戶
1.  **下載**：訪問 [python.org](https://www.python.org/downloads/) 下載最新的 Windows installer。
2.  **安裝 (至關重要的一步)**：
    在安裝介面的底部，**務必勾選 "Add Python.exe to PATH"**（將 Python 添加到環境變數）。
    > ⚠️ **警告**：如果不勾選這一項，你在終端機裡輸入 `python` 將沒有任何反應，後續所有操作都會受阻。
3.  **完成**：點擊 Install Now。

## 2. 什麼是 REPL？

**REPL** 是一個縮寫，代表 **R**ead（讀）、**E**val（算）、**P**rint（印）、**L**oop（循環）。

通俗地說，它就是 Python 的**「即時聊天模式」**。
在沒有圖形介面（像 LINE 那樣）的情況下，REPL 讓我們能直接和 Python 引擎對話。你問一句，它答一句。

---

## 3. 建立連接：呼叫解譯器

在終端機輸入啟動暗號：

*   **Mac**: `python3`
*   **Win**: `python`

當你看到 **`>>>`** 這個符號出現時，恭喜你，你已經成功「黑入」了 Python 的內部。
*此時，你已經暫時離開了 Shell 的管轄，進入了 Python 的領域。*

---

## 4. 體驗時刻：超級計算機

在系統學習程式語法（我們將在 **Module 2** 中詳細展開）之前，我們先來感受一下 Python 的原生能力。此時此刻，把它當成一個**地表最強計算機**。

嘗試輸入以下算式並按下 Enter：

### 4.1 基礎運算
```python
>>> 1 + 1
2
>>> 123 * 456
56088
```

### 4.2 指數爆炸
試試計算 2 的 100 次方（這在普通計算機上可能會報錯）：
```python
>>> 2 ** 100
1267650600228229401496703205376
```
這就是 Python 的魅力之一：它能輕鬆處理天文數字。

## 5. 初護記憶：變數 (Variables)

除了計算，Python 還能「記住」東西。

```python
>>> a = 10
>>> b = 20
>>> a + b
30
```

> **📚 預告**：
> 這裡發生的魔法叫做「變數賦值」和「記憶體管理」。
> **不用擔心現在聽不懂這些術語**。在後續的 **Module 2** 中，我們會系統性地拆解每一個概念。
> 現在，你只需要知道：**Python 有腦子，能記住你給它的數。**

---

## 6. 機制體驗：REPL vs 腳本

如果你現在關掉終端機視窗（點擊紅叉），或者輸入 `exit()` 退出。
再次進入 Python REPL，輸入 `a`，你會發現：
```python
>>> a
Traceback (most recent call last):
  NameError: name 'a' is not defined
```
**它失憶了！**

這是 REPL 最大的特點（也是缺點）：**它是臨時的**。
*   它適合做實驗、打草稿。
*   它**不適合**寫長久的程式。

**思考**：如果我們想寫一個永久執行的 AI 程式，該怎麼辦？
這就是 **M1L5** 要講的內容：如何把這對話「寫進檔案裡」保存下來。

## 7. 工程自檢 (Checkpoint)

### ✅ 驗收標準
1.  **環境就緒**：終端機執行 `python --version` 能顯示版本。
2.  **進入 Python**：看到 `>>>` 提示字元。
3.  **計算測試**：成功計算 `2 ** 10` (1024)。
4.  **安全退出**：使用 `exit()` 或快速鍵 (`Ctrl+D` Mac / `Ctrl+Z` Win) 回到終端機 Shell。

### 🛑 常見故障
*   **輸入 `python` 沒反應 (Windows)**：必定是安裝時沒勾選 "Add to PATH"。請卸載重裝，並務必勾選。
*   **在 `>>>` 裡輸入 `ls` 報錯**：記得嗎？這裡是 Python，不是 Shell。你需要先退出 (`exit()`) 才能用系統指令。
