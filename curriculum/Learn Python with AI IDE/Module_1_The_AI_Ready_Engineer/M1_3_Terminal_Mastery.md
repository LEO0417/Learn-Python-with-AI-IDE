---

## 1. 理论：终端是程序员的“母语”

在第一课中，我们提到要拒绝做“开箱者”。而终端 (Terminal) 就是你撕开黑盒的第一把手术刀。

### 1.1 为什么不点击鼠标？
鼠标操作（GUI）是直观的，但它是**不可自动化的**。
终端操作（CLI）是**高度精确且可编程的**。
当你掌握了终端，你就可以通过一行咒语批量处理 1000 个文件，或者在远程服务器（比如斯坦福后续课程中的 GPU 训练端）上瞬间部署一个系统。

### 1.2 Shell 与内核
*   **Kernel (内核)**：计算机的“心脏”，负责直接控制 CPU、内存和硬盘。它只听得懂电信号。
*   **Shell (壳)**：包在内核外面的一层。它负责接收你输入的文字（如 `ls`），翻译成内核懂的操作，再把结果返回给你。
**这也是为什么我们叫它“壳”。**

## 2. 战场选择：Zsh vs PowerShell

不同的操作系统，内置了不同的“翻译官”。

### 2.1 macOS: Zsh (Unix 标准)
Mac 默认使用的 Shell 叫做 **Zsh**（几年前是 Bash）。它是基于 Unix 哲学的，所有的命令都非常简洁、短促。
*   **特征**：区分大小写 (`A.txt` 和 `a.txt` 是两个文件)。
*   **哲学**：一切皆文件 (Everything is a file)。

### 2.2 Windows: PowerShell (面向对象)
Windows 提供的是 **PowerShell**。
*   **特征**：通常不区分大小写。
*   **优势**：它是现代的，支持把运行结果作为“对象”来处理。
*   **兼容性**：PowerShell 的开发者为了照顾从 Mac/Linux 转过来的用户，给很多命令起了别名 (Alias)。比如在 Windows 里打 `ls`，其实它运行的是 `Get-ChildItem`，效果一样。

## 3. 维度感：绝对路径 vs 相对路径

这是新手最容易迷路的地方。理解了路径，你就理解了计算机的文件层级。

### 3.1 绝对路径 (Absolute Path)
它是从系统的根部开始的完整路线图。
*   **macOS**: `/Users/leowang/projects/ai_course` (以 `/` 开头)。
*   **Windows**: `C:\Users\leowang\projects\ai_course` (以驱动器盘符开头)。
**无论你在哪里，绝对路径永远有效。**

### 3.2 相对路径 (Relative Path)
它是相对于你“当前位置”的路线图。
*   `.`：表示当前文件夹。
*   `..`：表示上一级文件夹。
*   `cd ./data`：进入当前目录下的 data 文件夹。
**相对路径效率更高，它是代码中写死路径的大忌（除非你是为了让项目在他人电脑也能运行）。**

## 4. 深度指令集 (Command Suite)

请打开终端，务必保证你已经激活了 `(ai_course)` 环境。

### 4.1 罗盘方位：`pwd`
*   **含义**: Print Working Directory.
*   **硬核点**: 在后续的程序开发中，很多报错是因为你的 `pwd` 不在项目的根目录。

### 4.2 扫描视野：`ls` (mac) / `dir` (win)
*   `ls -a`: 显示隐藏文件（以 `.` 开头的文件，通常是重要的配置文件）。
*   `ls -l`: 显示详细信息（谁创建的、多大、什么时候修改的）。

### 4.3 瞬间移动：`cd`
*   `cd ~`: 快速回到“家”目录（用户根目录）。
*   `cd -`: 回到上一次所在的目录（像遥控器的“回看”键）。

### 4.4 创造与清理：`mkdir` & `rm`
*   `mkdir -p a/b/c`: 递归创建路径（一次性建好爷爷文件夹、爸爸文件夹、孙子文件夹）。
*   `rm -rf folder_name`: 
    > [!CAUTION]
    > **这是一把极其锋利的双刃剑。** `-r` 表示递归删除文件夹，`-f` 表示强制删除不提示。
    > **警告：在没有确认路径前，绝对不要乱用 `rm -rf`。**

## 5. 指管分离：终端快捷键 (Power Moves)

真正的工程师不怎么按退格键：
*   **Tab 键**：**自动补全**。输入文件夹的前几个字母然后按 Tab，终端会自动填满。这是防止输入错误最高级的手段。
*   **方向键 ↑/↓**：调出历史命令。
*   **Ctrl + C**：强行中止。如果你的程序陷入了死循环，按它救命。
*   **Cmd + K (Mac) / Ctrl + L (Win)**：快速清屏，还你清爽视野。

## 6. 硬核练习

1.  **迷宫构建大挑战**：
    在你的桌面 (`Desktop`) 下，一次性创建一个三层嵌套的目录：`lab/source/code`。
    然后，使用一行命令在 `code` 文件夹里创建一个空文件 `hello.py`（提示：Mac 用 `touch`，Win 用 `New-Item` 或搜索 AI 如何创建空文件）。
2.  **坐标瞬移**：
    从 `code` 文件夹里，仅用一个 `cd` 命令和相对路径定位方式，跳回到 `lab` 文件夹，并用 `pwd` 确认。
3.  **隐藏档案发现**：
    在你的用户根目录下运行 `ls -a`。数数有多少个以 `.` 开头的文件或目录。去搜一搜 `.bash_history` 或 `.zsh_history` 是干什么用的。
