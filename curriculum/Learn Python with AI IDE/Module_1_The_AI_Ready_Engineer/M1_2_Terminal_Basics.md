## 1. 什么是终端 (Terminal)？

在你习惯的 Windows 或 macOS 图形界面 (GUI) 出现之前，人类就是通过黑底白字的屏幕与计算机交流的。
这个黑框框通常被称为 **Terminal**（终端），运行在里面的程序叫做 **Shell**（壳）。

*   **Kernel (核)**：计算机的核心，管理硬件和资源。
*   **Shell (壳)**：包在核外面的一层，负责把你的指令翻译给核听。

掌握终端，意味着你绕过了图形界面的限制，拥有了直接操控计算机的最高权限。

## 2. 打开终端

### macOS 用户
1.  按下 `Cmd + Space` 呼出聚焦搜索。
2.  输入 `Terminal` 并回车。
3.  你会看到一个白底黑字（或黑底白字）的窗口，上面有个光标在闪烁。

### Windows 用户
1.  按下 `Win` 键。
2.  输入 `PowerShell` 并回车。
3.  你会看到一个蓝底白字的窗口。

> **注意**：在本课程中，我们将统一使用类 Unix 的命令风格。Windows 的 PowerShell 大部分指令与 macOS 兼容。

## 3. 核心指令：导航 (Navigation)

想象你在一个巨大的文件柜迷宫里，全黑，你手里只有一个手电筒。你需要通过指令来确认位置和移动。

### 3.1 `pwd`: 我在哪里？
*   全称：**P**rint **W**orking **D**irectory (打印当前工作目录)
*   输入：`pwd` 并回车。
*   输出：例如 `/Users/leowang`。这是你当前的坐标。

### 3.2 `ls`: 这里有什么？
*   全称：**L**i**s**t (列出)
*   输入：`ls` 并回车。
*   输出：你会看到当前目录下的所有文件和文件夹的名字。

### 3.3 `mkdir`: 建造新的房间
*   全称：**M**a**k**e **Dir**ectory (创建目录/文件夹)
*   输入：`mkdir my_python_project` 并回车。
*   结果：虽然没有文字输出，但如果你再运行一次 `ls`，你会发现多了一个 `my_python_project` 文件夹。

### 3.4 `cd`: 瞬间移动
*   全称：**C**hange **D**irectory (改变目录)
*   输入：`cd my_python_project` 并回车。
*   再输入 `pwd`，你会发现你的坐标已经变了。
*   **回家指令**：输入 `cd ..` (注意中间有空格)，回到上一级目录。

### 3.5 `clear`: 清理视野
*   当屏幕上文字太多时，输入 `clear`，还你一个干净的工作台。

## 4. 练习 (Practice)

请在终端中依次完成以下操作：
1.  打印当前路径 (`pwd`)。
2.  列出当前文件 (`ls`)。
3.  创建一个名为 `demo` 的文件夹 (`mkdir demo`)。
4.  进入 `demo` 文件夹 (`cd demo`)。
5.  确认当前位置 (`pwd`)。
6.  回到上一级 (`cd ..`)。

完成这套动作，你就已经掌握了程序员 50% 的日常操作了。
