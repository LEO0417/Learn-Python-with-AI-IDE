在上一课 [M1L2] 中，我们掌握了终端的使用。现在，我们将用终端做一件真正像程序员做的事：唤醒 Python。

对于初学者来说，Python 不仅仅是一门编程语言，通过 **REPL**（Read-Eval-Print Loop），它更像是一个随叫随到的“超级助手”。你不需要写长篇大论的代码，只需要像聊天一样发一条指令，它就会立刻回复你。

## 1. 第一步：环境自检与安装

在开始之前，我们需要确认你的电脑里是否已经住着这只“巨蟒”。

### 1.1 检查是否存在
打开终端，输入：
```bash
# Mac 用户：
python3 --version

# Windows 用户：
python --version
```
*   **成功**：看到 `Python 3.x.x`。
*   **失败**：`command not found` 或无反应。

### 1.2 安装指南 (如果没有)

#### 🍎 macOS 用户
1.  **推荐方法**：访问 [python.org](https://www.python.org/downloads/) 下载最新的 `macOS 64-bit universal2 installer`。
2.  **安装**：双击 `.pkg` 文件，一路“继续”。
3.  **关键动作**：安装完成后，**必须关闭并重启终端**，才能让新安装生效。

#### 🪟 Windows 用户
1.  **下载**：访问 [python.org](https://www.python.org/downloads/) 下载最新的 Windows installer。
2.  **安装 (至关重要的一步)**：
    在安装界面的底部，**务必勾选 "Add Python.exe to PATH"**（将 Python 添加到环境变量）。
    > ⚠️ **警告**：如果不勾选这一项，你在终端里输入 `python` 将没有任何反应，后续所有操作都会受阻。
3.  **完成**：点击 Install Now。

---

## 2. 什么是 REPL？

**REPL** 是一个缩写，代表 **R**ead（读）、**E**val（算）、**P**rint（印）、**L**oop（循环）。

通俗地说，它就是 Python 的**“即时聊天模式”**。
在没有图形界面（像微信那样）的情况下，REPL 让我们能直接和 Python 引擎对话。你问一句，它答一句。

---

## 3. 建立连接：呼叫解释器

在终端输入启动暗号：

*   **Mac**: `python3`
*   **Win**: `python`

当你看到 **`>>>`** 这个符号出现时，恭喜你，你已经成功“黑入”了 Python 的内部。
*此时，你已经暂时离开了 Shell 的管辖，进入了 Python 的领域。*

---

## 4. 体验时刻：超级计算器

在系统学习编程语法（我们将在 **Module 2** 中详细展开）之前，我们先来感受一下 Python 的原生能力。此时此刻，把它当成一个**地表最强计算器**。

尝试输入以下算式并回车：

### 4.1 基础运算
```python
>>> 1 + 1
2
>>> 123 * 456
56088
```

### 4.2 指数爆炸
试试计算 2 的 100 次方（这在普通计算器上可能会报错）：
```python
>>> 2 ** 100
1267650600228229401496703205376
```
这就是 Python 的魅力之一：它能轻松处理天文数字。

---

## 5. 初识记忆：变量 (Variables)

除了计算，Python 还能“记住”东西。

```python
>>> a = 10
>>> b = 20
>>> a + b
30
```

> **📚 预告**：
> 这里发生的魔法叫做“变量赋值”和“内存管理”。
> **不用担心现在听不懂这些术语**。在后续的 **Module 2** 中，我们会系统性地拆解每一个概念。
> 现在，你只需要知道：**Python 有脑子，能记住你给它的数。**

---

## 6. 机制体验：REPL vs 脚本

如果你现在关掉终端窗口（点击红叉），或者输入 `exit()` 退出。
再次进入 Python REPL，输入 `a`，你会发现：
```python
>>> a
Traceback (most recent call last):
  NameError: name 'a' is not defined
```
**它失忆了！**

这是 REPL 最大的特点（也是缺点）：**它是临时的**。
*   它适合做实验、算草稿。
*   它**不适合**写长久的程序。

**思考**：如果我们想写一个永久运行的 AI 程序，该怎么办？
这就是 **M1L5** 要讲的内容：如何把这对话“写进文件里”保存下来。

---

## 7. 工程自检 (Checkpoint)

### ✅ 验收标准
1.  **环境就绪**：终端运行 `python --version` 能显示版本。
2.  **进入 Python**：看到 `>>>` 提示符。
3.  **计算测试**：成功计算 `2 ** 10` (1024)。
4.  **安全退出**：使用 `exit()` 或快捷键 (`Ctrl+D` Mac / `Ctrl+Z` Win) 回到终端 Shell。

### 🛑 常见故障
*   **输入 `python` 没反应 (Windows)**：必定是安装时没勾选 "Add to PATH"。请卸载重装，并务必勾选。
*   **在 `>>>` 里输入 `ls` 报错**：记得吗？这里是 Python，不是 Shell。你需要先退出 (`exit()`) 才能用系统命令。
