## 3. 🏗️ Core Meaning (核心邏輯/機制)

### 3.1 Module 1 知識大圖景
我們將這 10 課串聯起來，你會發現你已經構建了一個完整的**「現代化工程系統」**：

1.  **世界觀** (L1)：理解 AI 時代為什麼還要學程式碼（主權與思維）。
2.  **根基** (L2-L3)：掌握**終端機**和 **REPL**，學會與機器直接對話。
3.  **防護** (L4)：學會用 **Conda** 建立虛擬實驗室，拒絕環境污染。
4.  **持久化** (L5)：從即時對話轉向**腳本編寫**。
5.  **武裝** (L6-L7)：配置 **AI IDE** 並建立**標準工作台**，讓開發井然有序。
6.  **協作** (L8-L9)：駕馭 **AI Sidebar**，構建 **Human-AI Loop**。

### 3.2 工程師的「肌肉記憶」
真正的工程師不是背誦者，而是擁有「肌肉記憶」的實踐者。當你遇到問題能本能地打開終端機、查看路徑、喚起 AI、閱讀報錯時，你已經脫胎換骨了。

## 4. 🛠️ Practical Practice (操作與練習)

### 終極自測：你的工作台合格嗎？
請檢查你電腦上的 `my_first_project` 資料夾：
1.  **結構檢查**：是否有清晰的目錄？是否包含 `README.md`？
2.  **環境檢查**：在終端機輸入 `conda info --envs`，你是否能看到 `ai_course`？
3.  **執行檢查**：在 IDE 的終端機中執行 `python hello.py`，是否能成功輸出結果？
4.  **協作檢查**：嘗試對 AI 說：「幫我給現在的 README 增加一個『通關感言』章節」，並使用 **Apply** 合入。

-   **建立你的 Snippets（程式碼片段）**：把常用的指令或 Prompt 存起來。
