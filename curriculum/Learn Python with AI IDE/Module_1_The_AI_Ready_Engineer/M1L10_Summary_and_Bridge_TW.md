在過去的九課中，你已經完成了從「編程新手」到「AI 準備就緒的工程師」的第一次蛻變。讓我們看看你已經掌握了哪些核心基座：

*   **L1-L2：物理世界認知**。你掌握了**終端機 (Terminal)** 的使用，學會了目錄跳轉 (`cd`)，看清了電腦檔案系統的本質。
*   **L3-L4：環境隔離意識**。你理解了為什麼不能污染「毛坯房」，並成功建立了專屬於本課程的 **`ai_course` 虛擬環境**。

*   **L8-L9：人機協作進化**。你激活了 **Toggle Agent**，掌握了上下文引用 (`@`) 和 **Review-Refine-Run** 循環，確立了你作為「指揮官」的地位。

## 2. 啟航 Module 2：Python 核心語法

恭喜你！你已經搭建好了最先進的「數位工作站」。

在此我們需要停頓一下，請你意識到這一成就的含金量：**你現在所擁有的 `scripts` 資料夾，不再只是一個存放檔案的地方。它處於一個被 Python 插件實時監聽、被 `ai_course` 虛擬環境隔離保護、且被 Antigravity 的 AI 引擎深度索引的「工業級開發環境」。** 許多職業開發者在入行初期，也要花很久才能理清這些底層邏輯。

現在，我們正式開始學習如何駕馭這個強大的工具了。

在接下來的 **Module 2: Python Essentials** 中，我們的學習方式將發生翻天覆地的變化：

### 2.1 學即執行 (Interactive Learning)
我們不再只是枯燥地背單詞。在 Module 2 中，每當你學到一個新概念（變數、列表、循環），我們都會立刻在 **Antigravity** 中新建一個腳本，並由 AI 輔助你執行它、玩轉它。

### 2.2 你將掌握的工程基因
*   **資料結構**：學習如何讓電腦記住你給它的資訊。
*   **邏輯控制**：學習如何讓代碼具備「思考」和「分支判斷」的能力。
*   **函數與模組**：學習如何將雜亂的代碼封裝成強大的工程組件。
