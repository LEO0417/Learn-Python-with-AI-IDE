# Module 1 Lesson 2: 工程師的第一語言：終端機 (Terminal)

> **課程體系：Learn Python with AI IDE**
> **當前模組**：Module 1 - The AI Ready Engineer
> **本課目標**：克服對「黑框框」的恐懼，學會打開終端機，掌握最基礎的導航與檔案操作。

---

在本課程的開篇，我們提到工程師要掌握「底層邏輯」。對於很多初學者來說，電腦螢幕上那個黑壓壓的視窗（終端機）就像是駭客電影裡的東西，讓人望而生畏。

**為什麼要學這個？理由有兩個：**

1.  **它是 Python 的啟動器**：你無法用滑鼠雙擊打開一個 `.py` 程式碼檔案來執行它。在真正的開發中，我們必須透過終端機輸入指令來啟動 Python 引擎。
2.  **它是 AI 操作電腦的手**：當你對 AI 說「幫我執行這個專案」時，AI 實際上是在後台為你輸入終端機指令。如果你看不懂這些指令，你就無法知道 AI 到底在對你的電腦做什麼（比如刪除檔案或上傳資料）。

**掌握終端機，你才能看懂 AI 的操作，真正成為能夠「審查」和「指揮」AI 的工程師。**

## 1. 第一步：找到並打開終端機

我們要做的第一件事，就是找到並打開這個程式。它一直躲在你電腦的角落裡。

### 🍎 macOS 用戶
1.  按下鍵盤上的 **Command (⌘) + 空格**，叫出 Spotlight 搜尋框。
2.  輸入 `Terminal`（或者中文「終端機」）。
3.  點擊那個黑色的圖示。
4.  如果你想要更現代的體驗，以後可以下載 `iTerm2`，但現在內建的就足夠了。

### 🪟 Windows 用戶
1.  按下鍵盤上的 **Windows 標誌鍵**。
2.  直接輸入 `PowerShell`。
3.  **注意**：不要用那個叫 `CMD` (命令提示字元) 的老古董，它太舊了。請認準 **PowerShell**。

---

## 2. 第二步：你好，作業系統

打開後，你會看到一行奇怪的字元（比如 `User@MacBook`），結尾通常是一個符號（Mac 是 `$` 或 `%`，Windows 是 `>`）。

這叫做 **提示字元 (Prompt)**。它後面的那個閃爍游標，就是在告訴你：「我在聽，請下令」。

讓我們來嘗試第一次對話。輸入以下指令，然後按下 **Enter**：

```bash
echo "Hello World"
```

如果螢幕上回覆了 `Hello World`，恭喜你！你剛剛成功透過純文字向作業系統下達了指令並收到了回饋。

---

## 3. 第三步：我在哪裡？（定位與偵察）

這是新手最容易迷路的地方。在圖形介面裡，你一眼就能看到自己在「桌面」還是「下載」資料夾。但在終端機裡，你需要主動詢問。

請跟著我做：

### 3.1 詢問座標：`pwd`
輸入 `pwd` 並按下 Enter。
*   意思是：**P**rint **W**orking **D**irectory (列印當前工作目錄)。
*   你會看到類似 `/Users/leowang` (Mac) 或 `C:\Users\leowang` (Win) 的回覆。
*   這就代表你現在的「腳」踩在哪個資料夾裡。通常預設打開時，你都在**家目錄 (Home)**。
*   **注意**：在終端機的世界裡，我們通常把這個地址叫做**「路徑」 (Path)**，而不是簡單的「資料夾」。

### 3.2 環顧四周：`ls`
輸入 `ls` 並按下 Enter。
*   意思是：**L**i**s**t (列出清單)。
*   你會看到 `Desktop` (桌面), `Downloads` (下載), `Documents` (文件) 等等。
*   這相當於你在檔案總管裡看到了當前資料夾下的所有東西。

---

## 4. 第四步：瞬間移動（導航）

現在我們想去「桌面」看看。

### 4.1 進入資料夾：`cd`
輸入 `cd Desktop` 並按下 Enter。
*   意思是：**C**hange **D**irectory (改變目錄)。
*   再次輸入 `pwd`，你會發現路徑變長了，結尾變成了 `/Desktop`。

### 4.2 回家：`cd ~`
不管你走到了多深的資料夾裡，只要你想回家：
*   輸入 `cd ~` (波浪號) 並按下 Enter。
*   或者直接輸入 `cd` (不帶任何參數) 並按下 Enter。
*   再次 `pwd`，你會發現自己又回到了 `/Users/leowang`。

> [!TIP]
> **新手救生圈**：如果你不知道檔名怎麼拼，打前幾個字母，然後按 **Tab 鍵**！電腦會自動幫你補完。如果按 Tab 沒反應，說明你打錯了或者檔案不存在。

---

## 5. 第五步：上帝之手（建立與整理）

現在我們有了地圖，也有了移動能力，讓我們試著造點東西。

### 5.1 建立資料夾：`mkdir`
確保你在家目錄（`cd ~`）。
輸入 `mkdir my_ai_project` 並按下 Enter。
*   意思是：**M**a**k**e **Dir**ectory (製造目錄)。
*   用 `ls` 看一下，是不是多了一個 `my_ai_project`？去你的圖形化檔案總管裡看看，它也一定在那裡。

### 5.2 建立檔案：`touch`
先進入剛才的資料夾：`cd my_ai_project`。
輸入 `touch first_code.py` 並按下 Enter。
*   *Windows 用戶如果 `touch` 報錯，請輸入 `new-item first_code.py`，或者直接用 `echo "" > first_code.py`。*
*   這是一個在 Mac/Linux 極其常用的命令，用於「摸」出一個空檔案。

## 6. 進階理解：路徑的奧秘

在上面我們簡單接觸了路徑，現在讓我們像工程師一樣深入理解它。路徑是檔案系統的座標系。

### 6.1 四個特殊的符號
*   **`~` (波浪號)**：**Home（家目錄）**。
    *   這是你的私人領地。無論你在哪裡，輸入 `cd ~` 瞬間回家。
    *   Win: `C:\Users\YourName` | Mac: `/Users/YourName`
*   **`/` (斜線)**：**Root（根目錄）**。
    *   這是電腦檔案系統的總源頭。所有檔案都掛在這個樹根下。
    *   *警告*：新手請勿在 `/` 下隨意刪除檔案，那裡是作業系統的地盤。
*   **`..` (雙點)**：**Parent（父目錄）**。
    *   「撤退」指令。代表上一層資料夾。
    *   `cd ..` = 退一層。 `cd ../..` = 退兩層。
*   **`.` (單點)**：**Current（當前位置）**。
    *   代表「就在這裡」。通常用於執行當前目錄下的程式（如 `./script.sh`）。

### 6.2 絕對路徑 vs 相對路徑
*   **絕對路徑 (Absolute)**：以 `/` 或磁碟代號開頭。
    *   `/Users/leowang/Desktop/Lab1`
    *   特點：**地址是死的**。無論你在哪裡輸入，它都指向同一個地方。就像 GPS 座標。
*   **相對路徑 (Relative)**：**不**以 `/` 開頭。
    *   `Desktop/Lab1`
    *   特點：**地址是活的**。它取決於你現在站在哪裡。就像說「往前走」。

> [!CAUTION]
> **常見大坑：空格**
> 如果你的資料夾名叫 `My Project`（中間有空格），終端機會以為這是兩個資料夾：`My` 和 `Project`。
> *   **解法 1**：用引號包起來 -> `cd "My Project"`
> *   **解法 2 (推薦)**：養成好習慣，**檔名永遠不要用空格**，用底線 `_` 代替 -> `My_Project`。

## 7. 生存技巧：像工程師一樣思考

在這一節，我們將不僅是操作，而是要理解你和 AI 交流的「文法」。

### 7.1 拒絕重複：歷史紀錄 (History)
工程師最討厭做重複的事。如果你剛剛敲了一長串命令，發現有一個字母拼錯了，**千萬不要重新打一遍**。
*   **動作**：按一下鍵盤上的 **⬆️ (上箭頭)**。
*   **效果**：你會發現上一條命令自動出現了。按幾次就是往回翻幾次。

### 7.2 讀懂 AI 的咒語：指令解剖學
有時候 AI 會給你這樣的指令：
```bash
rm -rf my_project
```
新手看這行程式碼會覺得是亂碼，但工程師看到的是一個清晰的句子結構：**動詞 + 形容詞 + 名詞**。

*   **Command (動詞)**: `rm` (Remove, 刪除)。要做什麼？
*   **Flag/Option (形容詞)**: `-rf`。怎麼做？
    *   `-r`: (Recursive) 遞迴地，連資料夾裡的東西一起刪。
    *   `-f`: (Force) 強制地，不要每次都問我確不確定。
*   **Argument (名詞)**: `my_project`。對誰做？

**練習**：對比 `ls -a`。
*   `ls` 是動作（看）。
*   `-a` 是形容詞（全部，包括隱藏檔案）。

### 7.3 後悔藥：緊急煞車 (Ctrl + C)
如果你不小心讓電腦進入了無窮迴圈（比如瘋狂列印螢幕），或者安裝卡住了。
*   **動作**：按下 **Ctrl + C**（Mac 和 Windows 都是這個）。
*   **效果**：強制終止當前程式。這是你的安全繩。

## 8. 幕后機制：Shell 與 Terminal

最後，作為工程師，你需要知道你剛才用的那個黑框框到底是什麼：

1.  **Terminal (終端機)**：就是這個黑色的視窗軟體。它只是個顯示器。
2.  **Shell (殼/解譯器)**：視窗背後那個處理你文字的「大腦」。
    *   Mac 預設用的是 **Zsh**。
    *   Windows 預設用的是 **PowerShell**。

當你輸入 `ls` 時，是 **Shell** 在系統裡找到了 `ls` 這個小程式，執行它，然後把結果顯示在 **Terminal** 上。

## 9. 工程自檢 (Checkpoint)

### ✅ 驗收標準
請像做遊戲任務一樣完成以下流程：
1.  **啟動**：成功打開 Terminal 或 PowerShell。
2.  **定位**：輸入 `pwd` 確認自己在 User 家目錄。
3.  **行動**：建立一個叫 `Lab1` 的資料夾，並 `cd` 進去。
4.  **創造**：在裡面建一個空檔案 `test.txt`。
5.  **撤退**：用 `cd ..` 退回家目錄。

### 🛑 常見坑點
*   **檔名有空格**：盡量不要用空格，用底線 `_`。如果非要有空格，用引號包起來：`cd "My Folder"`。
*   **Windows 也有 ls 嗎？**：新的 PowerShell 有，但老舊的 CMD 只能用 `dir`。這就是為什麼我們要用 PowerShell。
