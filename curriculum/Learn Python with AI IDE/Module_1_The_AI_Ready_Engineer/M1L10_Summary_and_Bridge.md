## 2. 衔接 Module 2：开启 Python 语法征途

现在，你已经搭建起了一个受解释器监听、受环境隔离且被 AI 引擎深度索引的**标准工程环境**。这是进行后续系统化编程学习的最优选择。

从 **Module 2: Python Essentials** 开始，我们将正式进入编程逻辑的学习：

### 2.1 实践式驱动 (Lab-based Learning)
在 Module 2 中，我们不再孤立地记忆语法规则。每当你接触到一个新概念（如参数、循环或对象），你都将立刻在 **Antigravity** 中创建对应的实验脚本，并在 AI 辅助下快速运行验证。

### 2.2 即将掌握的核心能力
*   **数据结构**：学习如何让代码管理复杂的信息。
*   **逻辑流控制**：学习如何让程序具备分支判断与重复执行的能力。
*   **封装与重用**：学习如何编写函数（Function）与模块（Module），构建可扩展的程序。
