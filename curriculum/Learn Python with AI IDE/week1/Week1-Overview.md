# Learning Python with AI IDE Week1：周总览

## 本周学习理念
- 我们不是把 AI 当“替身”，而是把 AI 当“被授权的助手”。
- 你必须懂基础，才能判断 AI 是否做对、做错、做偏。
- Week1 的目标是先建立“可运行、可复现、可解释”的基础闭环。

## 本周主线目标
- 会用终端完成最基本的定位与操作。
- 会在终端里搭建并激活 Python 环境。
- 会在 IDE 里写脚本、运行脚本、看报错。
- 会把“终端 + 环境 + IDE + 脚本”完整串起来。
